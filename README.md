# animegirlsholdingprogrammingbooks.com

A modern front-end for browsing the [Anime Girls Holding Programming Books](https://github.com/cat-milk/Anime-Girls-Holding-Programming-Books) repository with infinite scroll gallery.

All images and the original collection belong to the [cat-milk](https://github.com/cat-milk) repository. This project is an independent front-end only.
